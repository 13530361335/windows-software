**Linux命令:**
```
cat  /proc/cpuinfo       查看cpu
free -h                  查看内存
df   -h                  查看磁盘
```

**IntelliJ IDEA快捷键:**
```
Ctrl + R                 搜索/替换
Ctrl + Shift + R         全局搜索/替换
alt  + Shift + Up/Down   上下行代码互换
alt  + left/right        切换代码视图
```

**Google Chrome快捷键:**
```
Ctrl + U                 查看源码
Ctrl + F5                刷新(忽略缓存)
Ctrl + Shift + delete    清除缓存
Ctrl + Tab(1~9)          切换标签
Ctrl + T                 打开标签
Ctrl + W                 关闭标签
```
